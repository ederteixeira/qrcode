# Gerador QR Code

## Descrição:

Gerador de QR code

    Geração da imagem em formato .PNG
    Geração da imagem em formato .SVG

#### Documentação: 

**https://pypi.org/project/PyQRCode/**

#### Tecnologia:
* Python 3

#### Bibliotecas:

* pyQrcode     

      pip3 install pyqrcode

* pyPng

      pip3 install pypng
